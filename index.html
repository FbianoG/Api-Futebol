É importante ressaltar que, devido à natureza da API utilizada, os dados fornecidos podem conter erros e inconsistências, e não é possível alterar a data das partidas ou corrigir manualmente quaisquer imprecisões.

Portanto, nossa aplicação foi desenvolvida levando em consideração essa limitação, garantindo que os usuários tenham uma experiência informacional transparente e precisa.
